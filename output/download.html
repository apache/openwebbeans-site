Apache OpenWebBeans Releases¶

This page contains download links to the latest Apache OpenWebBeans releases.

All maven artifacts are available in the Maven.Central repository with the groupId org.apache.openwebbeans. The dependencies you can use are listed at the bottom of this page: Maven Dependencies.

KEYS for verifying Apache releases¶

The GPG keys in the OpenWebBeans KEYS file to validate our releases. Read more about How to verify downloaded files

OWB-2.0.x¶

OWB-2.0.x implements the CDI-2.0 (JSR-365) specification. It uses a shaded version of ASM-9 (Java17 support) for building our proxies and requires JavaSE 8 as minimum version.

Source¶

Should you want to build any of the above binaries, this source bundle is the right one and covers them all.

  • openwebbeans-2.0.26-source-release.zip
  • openwebbeans-2.0.26-source-release.zip.sha512
  • openwebbeans-2.0.26-source-release.zip.asc

OWB-1.7.x¶

OWB-1.7.x implements the full CDI-1.2 specification. It uses a shaded version of ASM-5 for building our proxies and needs JavaSE 7 as minimum version.

Binaries¶

The binary distribution contains all OpenWebBeans modules.

  • openwebbeans-distribution-1.7.6-binary.zip
  • openwebbeans-distribution-1.7.6-binary.zip.sha1
  • openwebbeans-distribution-1.7.6-binary.zip.asc
  • openwebbeans-distribution-1.7.6-binary.tar.gz
  • openwebbeans-distribution-1.7.6-binary.tar.gz.sha1
  • openwebbeans-distribution-1.7.6-binary.tar.gz.asc

Source¶

Should you want to build any of the above binaries, this source bundle is the right one and covers them all.

  • openwebbeans-1.7.6-source-release.zip
  • openwebbeans-1.7.6-source-release.zip.sha1
  • openwebbeans-1.7.6-source-release.zip.asc

OWB-1.2.x¶

OWB-1.2.x implements the CDI-1.0 specification and internally already CDI-1.1. It uses a shaded version of ASM-5 for building our proxies and needs JavaSE 5 as minimum version.

Binaries¶

The binary distribution contains all OpenWebBeans modules.

  • openwebbeans-distribution-1.2.8-binary.zip
  • openwebbeans-distribution-1.2.8-binary.zip.sha1
  • openwebbeans-distribution-1.2.8-binary.zip.asc
  • openwebbeans-distribution-1.2.8-binary.tar.gz
  • openwebbeans-distribution-1.2.8-binary.tar.gz.sha1
  • openwebbeans-distribution-1.2.8-binary.tar.gz.asc

Hint: OpenWeBeans has dependencies to several other jars and just adding our jars manually would lead to ClassNotFoundException if you choose not to use maven. The jars you need depends on what modules you include. They are all contained in the binary distribution. We will try to add complete lists of this in the future, meanwhile please ask on the list or maybe look at the pom.xml for the modules you want to use.

Source¶

Should you want to build any of the above binaries, this source bundle is the right one covers them all.

  • openwebbeans-1.2.8-source-release.zip
  • openwebbeans-1.2.8-source-release.zip.sha1
  • openwebbeans-1.2.8-source-release.zip.asc

    <groupId>org.apache.geronimo.specs</groupId>

    <version>1.2</version>

<dependency>
    <groupId>org.apache.geronimo.specs</groupId>

</dependency>
