# OpenWebBeans Web


    <dependency>
        <groupId>org.apache.openwebbeans</groupId>
        <artifactId>openwebbeans-web</artifactId>
        <version>${owb.version}</version>
        <scope>compile</scope>
    </dependency>

Required for web-apps / Servlet-container projects that deploy to for example Tomcat.
